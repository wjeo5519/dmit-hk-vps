移动用户 → Eyeball（CMI 三网优化，移动体验接近 Pro）或 Pro 都行，T1 移动也凑合。电信/联通用户 → 不能妥协就选 Pro（CN2 GIA 才是真正保障），Eyeball 次之。

每月 $39.90 以上接受不了 → Pro 排除，Eyeball 起步；每月 $30 上下 → Eyeball TINYv2；不需要大陆速度，极致省钱 → T1 WEE 年付 $36.9 起，折合每月 $3。

说实话，对大多数个人用户来说，Eyeball 是最值得入手的一档：延迟比 T1 明显更低，价格比 Pro 少了 $10/月，流量还多一倍。除非你确实需要晚高峰极稳的 CN2 GIA 保障，Eyeball 够用了。
